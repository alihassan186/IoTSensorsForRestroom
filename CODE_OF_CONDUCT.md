# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive
- Welcome different viewpoints
- Give and receive feedback professionally
- Focus on what is best for the project

## Unacceptable Behavior

- Harassment, insults, or discriminatory language
- Trolling or personal attacks
- Publishing private information without permission
- Any conduct that is inappropriate in a professional setting

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions not aligned with this Code of Conduct.

## Reporting

Report unacceptable behavior to the maintainers via the process in `SECURITY.md`.
